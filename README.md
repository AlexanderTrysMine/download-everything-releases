# Download Everything releases

Official Windows companion packages for the Download Everything Chrome extension.

[![Install from Chrome Web Store](https://img.shields.io/badge/Install-Chrome%20Web%20Store-4285F4?logo=googlechrome&logoColor=white)](https://chromewebstore.google.com/detail/jnieflkienbnklgejgfhefamomcfanib)

The install button opens the Chrome Web Store listing. It becomes installable once Google approves and publishes the extension.

Download Everything is intended only for videos the user owns or is legally authorized to save. It does not bypass DRM, private access, memberships, or account restrictions.

Privacy and support: https://download-everything-support.michaelafelder7.chatgpt.site
